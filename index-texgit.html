JRimum Texgit

Texgit é um componente e framework em Java que facilita a manipulação de arquivos Flat Files (arquivo-texto) através de objetos.


Diversas empresas já usam o Texgit em produção e estão colaborando no Design do componente e da API. Embora a documentação ainda não tenha se desenvolvido, acreditamos que através de exemplos seu entendimento seja simples e claro.


Dependências

Obs: As versões das dependências aqui descritas são aquelas testadas e utilizadas pelo componente.

  • Java 1.5 ou superior.
  • JAXB 2.2 ou superior (Para o Java SE 5 o JAXB deve ser baixado separadamente. Já o Java SE 6 não precisa, pois o JAXB é embutido)
  • Apache Commons Lang (2.4) - Link direto ( Source)
  • Apache log4j (1.2.15) - Link direto ( Source)
